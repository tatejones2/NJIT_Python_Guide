# Functions

### Defining a function
```python
def add():
  a = int(input("Enter number: "))
  b = int(input("Enter number: "))
  return a + b
```

### Parameters
```python
def add(a, b):
  return a + b

add(6, 7) # output = 13
```

### Important functions
```python
print()    # outputs text or values
input()    # gets user input as a string
len()      # returns the length of a string, list, or other iterable
type()     # shows the data type of a value
int()      # converts to integer
float()    # converts to floating-point number
str()      # converts to string
range()    # generates a sequence of numbers, often used in loops
sum()      # adds up numbers in a list or iterable
max()      # gets the largest value
min()      # gets the smallest value
sorted()   # returns a sorted list
help()     # opens documentation for functions and objects
```
